# 🧱 TheFloorIsSomething

[![Minecraft Version](https://img.shields.io/badge/Minecraft-1.20%2B-brightgreen.svg)](https://www.spigotmc.org/)
[![Author](https://img.shields.io/badge/Author-XenonSpace-orange.svg)](https://github.com/XenonSpace)

**TheFloorIsSomething** is a professional, lightweight, and highly customizable Minecraft Spigot plugin. Every X seconds, a random block becomes "dangerous". If players stand on it, they take damage and catch fire!

## 🌟 Key Features

* **Massive Block Library:** Includes a paginated GUI menu containing every solid block in Minecraft.
* **Dynamic Customization:** Change the game timer and select active blocks directly in-game.
* **Immersive Experience:** Features high-quality sound effects, screen titles, and action bar warnings.
* **Persistent Storage:** All settings and enabled blocks are saved in a `config.yml` file.
* **Developer Friendly:** Clean Java code using the Spigot API 1.20.

## 🛠 Commands & Permissions

| Command | Description | Permission |
| :--- | :--- | :--- |
| `/floor help` | Displays the help menu | `floor.admin` |
| `/floor menu` | Opens the block selection GUI | `floor.admin` |
| `/floor time <sec>` | Sets the block rotation timer | `floor.admin` |

## 🚀 Installation

1.  Download the latest `.jar` from the [Releases](https://github.com/YOUR_GITHUB_USERNAME/TheFloorIsSomething/releases) page.
2.  Place the file in your server's `plugins` folder.
3.  Restart your server.
4.  Configure your blocks using `/floor menu` and start the chaos!

## 💻 Tech Stack
* **Language:** Java
* **API:** Spigot API (1.20.x)
* **Build Tool:** Maven
